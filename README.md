## Getting started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Requires

* Git
* Node

### Deploying development

**1.** Install dependencies:
```bash
npm i
```

**3.** Execute:
```bash
npm start
```
**4.** Open http://localhost:3000/
